# Лабораторная работа №1: Computer Vision (Классификация сортов риса)

## 1. Выбор начальных условий

### 1a) Набор данных
Использован **"Rice Image Dataset"**, состоящий из **75,000** изображений зерен риса пяти сортов (*Arborio, Basmati, Ipsala, Jasmine, Karacadag*). 

**Обоснование:** Задача автоматизированного контроля качества зерновых культур. Точная классификация позволяет автоматизировать сортировочные линии на производстве, исключая сортовое подмешивание и повышая скорость обработки продукции.

### 1b) Метрики качества
*   **Accuracy:** Общая точность классификации.
*   **F1-Score (Weighted):** Выбран как ключевая метрика, так как он учитывает баланс между точностью (Precision) и полнотой (Recall) для каждого сорта, что критично для промышленной сортировки.
*   **Confusion Matrix:** Использована для выявления конкретных пар сортов, которые модель путает из-за схожести геометрической формы.

---

## 2. Создание бейзлайна и оценка качества

На первом этапе обучение проводилось на ограниченной подвыборке (1200 изображений на класс) без использования техник аугментации данных.

**Результаты бейзлайна:**
*   **ResNet18 (CNN):** Accuracy/F1 — **88.0%**. Матрица ошибок выявила проблему: модель путала округлые сорта *Arborio* и *Karacadag* (Recall для Arborio составил всего 0.46).
*   **ViT-B/16 (Transformer):** Accuracy/F1 — **98.9%**. Трансформер показал лучший результат «из коробки», но требовал значительных вычислительных ресурсов.

---

## 3. Улучшение бейзлайна

### 3a) Гипотезы
1.  **Геометрическая аугментация:** Добавление `RandomHorizontalFlip` и `RandomRotation(15)` поможет моделям выучить инвариантные признаки формы зерен, независимо от их угла наклона в кадре.
2.  **Масштабирование данных:** Переход к обучению на полном наборе данных (**75,000 фото**) для минимизации риска переобучения и повышения обобщающей способности.

### 3b) Проверка и результаты
После внедрения аугментации и обучения на полном датасете (валидация на 15,000 фото) показатели значительно выросли:
*   **Улучшенный ResNet18:** Accuracy/F1 — **99.37%**.
*   **Улучшенный ViT-B/16:** Accuracy/F1 — **99.49%**.
Ошибки между схожими округлыми сортами были практически полностью устранены.

---

## 4. Имплементация алгоритма машинного обучения (Manual CNN)

### 4a) Самостоятельная реализация
Была спроектирована авторская архитектура **SimpleRiceNet**.
*   **Архитектура:** 3 сверточных слоя `Conv2d` (с увеличением числа фильтров 16-32-64), функции активации `ReLU`, слои `MaxPool2d` для уменьшения размерности и полносвязные слои с `Dropout` (0.2).
*   Модель реализована «с нуля» на PyTorch без использования предобученных весов.

### 4b) Результаты и сравнение
Собственная модель показала выдающиеся результаты после оптимизации:
*   **SimpleRiceNet (Базовая):** Accuracy/F1 — **98.66%**.
*   **SimpleRiceNet (С аугментацией):** Accuracy/F1 — **99.64%**.

**Итоговое сравнение:**
Самостоятельно имплементированная модель с аугментацией (**99.64%**) превзошла по точности предобученный ResNet18 и сравнялась с Vision Transformer. Это доказывает, что специализированная архитектура в сочетании с правильной подготовкой данных может быть эффективнее универсальных тяжелых моделей.
