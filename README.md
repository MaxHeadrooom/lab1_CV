# Лабораторная работа №1: Computer Vision (Классификация сортов риса)

## 1. Выбор начальных условий

### 1a) Набор данных
Был выбран набор данных **"Rice Image Dataset"**, содержащий 75,000 изображений зерен риса пяти различных сортов: *Arborio, Basmati, Ipsala, Jasmine, Karacadag*.

**Обоснование:** Это реальная практическая задача для агропромышленного сектора. Автоматизация сортировки риса позволяет исключить ошибки человеческого фактора, повысить скорость обработки продукции и гарантировать чистоту дорогостоящих сортов.

### 1b) Метрики качества
Для оценки моделей выбраны следующие метрики:
*   **Accuracy:** Основной показатель точности предсказаний.
*   **F1-Score (Weighted):** Для контроля того, что модель одинаково хорошо распознает все сорта (баланс точности и полноты).
*   **Confusion Matrix:** Для анализа того, какие именно сорта модель склонна путать между собой.

---

## 2. Создание бейзлайна

В качестве базовых моделей (бейзлайнов) из библиотеки `torchvision` были выбраны две архитектуры:
1.  **ResNet18 (CNN):** Классическая сверточная сеть.
2.  **ViT-B/16 (Transformer):** Современная архитектура на основе механизма внимания (Attention).

**Результаты бейзлайна:**
*   **ResNet18:** Accuracy на валидации — **85.8%**. Наблюдалось значительное переобучение (точность на трейне была выше 97%).
*   **ViT-B/16:** Accuracy на валидации — **98.9%**. Отличный результат «из коробки», но модель требует больших вычислительных ресурсов.

---

## 3. Улучшение бейзлайна

### 3a) Гипотезы
Был выдвинут ряд гипотез для улучшения результатов ResNet18:
1.  **Data Augmentation:** Добавление случайных поворотов (±15°) и горизонтальных отражений поможет модели выучить форму зерна, а не его положение в кадре.
2.  **Fine-Tuning LR:** Снижение шага обучения (Learning Rate) с 0.001 до 0.0005 для более плавной настройки весов.

### 3b-f) Проверка и результаты
После применения техник аугментации точность ResNet18 выросла до **99.8%**. Гипотеза полностью подтвердилась: аугментация эффективно устранила переобучение.

---

## 4. Имплементация алгоритма машинного обучения

### 4a) Самостоятельная реализация
Была спроектирована и реализована «с нуля» собственная сверточная нейросеть **SimpleRiceNet**.
*   **Архитектура:** 3 сверточных слоя (Conv2d) с функциями активации ReLU, слои MaxPool2d для уменьшения размерности и полносвязные слои (Linear) в конце.

### 4b-d) Результаты и сравнение
*   **SimpleRiceNet (Базовая):** Accuracy — **94.9%**.
*   **SimpleRiceNet (С аугментацией):** Accuracy — **97.5%**.

**Сравнение:** Самописная модель показала достойный результат, но уступила предобученным архитектурам (ResNet, ViT). Это объясняется отсутствием предварительного обучения на больших данных (ImageNet) и меньшей глубиной сети.
